# گل آرا — فروشگاه آنلاین گل

> ⚠ هشدار: این پروژه **در نسخه فعلی ریسپانسیو نیست** و ممکن است در صفحه‌نمایش‌های کوچک بهم‌ریخته دیده شود.

این پروژه یک فروشگاه آنلاین گل به اسم **گل آرا** است و پروژه پایانی دانشگاه من می‌باشد.

## نحوه مشاهده
- برای بهترین نتیجه از **Google Chrome** استفاده کنید.
- زوم را روی **100%** قرار دهید (Menu → Zoom → 100%).  
## ساختار صفحات

- `main.html` — صفحه اصلی فروشگاه  
- `productidpage/` — صفحات مشخصات هر محصول، هر `id` یک محصول متفاوت است  
- `productpage/` — صفحات دسته‌بندی محصولات  
- سایر صفحات — نام هر صفحه مشخص کننده محتوای آن است  

## تکنولوژی‌ها
- در این پروژه فقط از **HTML، CSS و JavaScript** استفاده شده است.  

## توضیحات تکمیلی
- این نسخه پروژه **ریسپانسیو نیست** و ممکن است در صفحات کوچک بهم‌ریخته شود.  
- نسخه حرفه‌ای‌تر و **ریسپانسیو شده** به زودی منتشر خواهد شد.  
# Gol Ara — Online Flower Shop

> ⚠ Warning: This project is **not responsive in its current version** and may appear broken on smaller screens.

This project is an online flower shop called **Gol Ara** and serves as my **final university project**.

---

## How to view
- For the best experience, use **Google Chrome**.
- Set the zoom level to **100%** (Menu → Zoom → 100%).

---

## Page structure

- `main.html` — Main shop page  
- `productidpage/` — Individual product detail pages; each `id` represents a different product  
- `productpage/` — Product category pages  
- Other pages — The page names indicate their content  

---

## Technologies used
- This project only uses **HTML, CSS, and JavaScript**.

---

## Additional notes
- This version of the project is **not responsive** and may appear broken on smaller screens.  
- A professional, **responsive version** will be released soon.
